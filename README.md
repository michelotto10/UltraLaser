# Ultra Personalizados

Projeto pessoal desenvolvido com foco em aprendizado prático em desenvolvimento web.

O site foi criado para divulgação de serviços e contato com clientes, aplicando conceitos estudados no curso técnico em Desenvolvimento de Sistemas.

 

---

##  Tecnologias Utilizadas
- HTML
- CSS
- PHP

---

##  Funcionalidades
- Páginas institucionais
- Estruturação de layout responsivo
- Área de contato para clientes
- Organização de arquivos e código

---

##  Objetivo do Projeto
Colocar em prática os conhecimentos adquiridos em sala de aula, especialmente em:
- Desenvolvimento web
- Estruturação de páginas
- Uso do PHP em aplicações simples
- Publicação de projeto em servidor real

---

##  Aprendizados
- Organização de um projeto web
- Uso de HTML e CSS para layout
- Aplicação de PHP em páginas dinâmicas
- Publicação e manutenção de site online
